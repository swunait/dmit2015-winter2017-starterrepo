DMIT2015 - Enterprise Application Development

DMIT2015 Starter Project

To use this project:

  1. Open Eclipse and import the GIT project from https://github.com/swunait/dmit2015-winter2017-starterrepo.git
  2. Rename the project (Refactor | Rename) to dmit2015-projectname-yourname
  3. In Project Explorer view, open "JPA Context |persistence.xml" and change the jta-data-source value to one of the following preconfigued wildfly datasources:
    jta-data-source Database
    java:jboss/datasources/ChinookDS Chinook
    java:jboss/datasources/northwindDS Northwind
    java:jboss/datasources/ClassicModelsDS ClassicModels
  4. Use the JSF Composition file "webapp | public | JSF-Composition-Page.xhtml" as a template when creating a new JSF web page.
  5. Customize the JSF template file "webapp | WEB-INF | templates | template-bootsfaces-starter.xhtml" to include new links to pages you created.
